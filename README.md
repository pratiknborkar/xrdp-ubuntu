# xrdp-ubuntu

 Update the list of available packages using the following command
 
```
sudo apt update
```

### Install the xfce and xfce-goodies packages

```
sudo apt install xfce4 xfce4-goodies -y
```

### Install xrdp 
```
sudo apt install xrdp -y
```
### Start xrdp service

```
sudo systemctl status xrdp
```
